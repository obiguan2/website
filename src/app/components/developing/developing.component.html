INTRODUCCIÓN

En los siguientes modelos realizaremos distintas formas de mostrar el modelamiento del sistema de venta y reserva de boletos, sin la necesidad de un cajero ya que sera online se podra modificar la cantidad de boletos y retirar la reserva igualmente.

Casos de uso de negocio y sistema

Descripción Permite abrir una cuenta de cliente de viaje la cual le permite realizar cualquiera operación
Flujo básico El cliente solicita la formulario de inscripción al vendedor o jefe en apertura de cuenta, este le entrega dicho documento para ser llenado, si la cuenta que se desea abrir no es invalida, el cliente deberá llenar en el formulario de inscripción los datos del otro cliente, después se le entregara el documento al especialista y este verificara con los documentos la cuenta que se va abrir no es individual o el documento si la cuenta que se va abrir es individual luego se genera user y contraseña o pin
Flujo alternatvo Si la verificación de formulario de inscripción fue incorrecta se rechaza dicha boleta y se le informa al cliente.

CURSO NORMAL DEL EVENTO

ACCIONES DEL ACTOR RESPUESTA DEL SISTEMA
1. El cliente ingresa al sitio web.
2.Despliega menú.
3. Selecciona la opción de Iniciar sesión.
4. Selecciona la opción de Nuevo Pasajero
5. Muestra interfaz de Nuevo Usuario.
6. Solicita los datos del usuario
7. Ingresa los datos como ser: nombres, apellidos, ci, correo.
8. Verifica los datos ingresados
9. Solicita la confirmación
10. Confirma los datos de usuario.
11. Despliega mensaje de aceptación.
12. Inicia sesión como pasajero. 13. Se habilita la opción de reserva y ventas.
